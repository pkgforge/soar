# Release Notes

This section contains release notes for Soar versions, documenting new features, breaking changes, and improvements.

- [Soar 0.12](./releases/v0.12.md): Repository consolidation (soarpkgs), repository management CLI, fuzzy search, metadata signature verification, local file installs, local metadata sources, and shell completions.
- [Soar 0.11](./releases/v0.11.md): Nest removal, custom desktop paths, and release notes on update.
- [Soar 0.10](./releases/v0.10.md): GitHub and GitLab sources, build from source, hooks, Landlock sandboxing, and system-wide installs.
